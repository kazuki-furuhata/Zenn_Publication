株式会社 MBK デジタルの古畑です。

この度、Google Cloud Japan が主催するアワードプログラム『Google Cloud Partner Top Engineer 2026』のGeneral部門に選出いただきました。
おかげさまで昨年に続き、2年連続での選出となります。 

日頃のプロジェクトやコミュニティ活動をこのような形で評価いただき、大変光栄に思います。
また、日頃サポートしてくれているチームメンバーや関係者の皆様に改めて感謝申し上げます。

会社よりプレスリリースが発表されましたので、本ブログでもその内容をご紹介させていただきます。

**【以下、プレスリリースより】**

## MBKデジタルのエンジニアが「Google Cloud Partner Top Engineer 2026」選出
Google Cloud 技術領域での取り組みが評価され、General 部門で認定

三井物産株式会社の100%子会社である株式会社MBKデジタル（本社：東京都千代田区、代表取締役社長：芹澤 新、以下MBKデジタル）は、このたび、グーグル・クラウド・ジャパン合同会社（Google Cloud Japan）が主催する Google Cloud Partner Top Engineer 2026 アワードプログラムにおいて、当社デジタルプラットフォーム事業本部プロダクト＆テクノロジー部データインテリジェンスグループ データエキスパートの古畑 和輝がGeneral部門に選出されたことをお知らせします。

![写真](/images/articles/93783fa8cc8555_1.png)

「Google Cloud Partner Top Engineer」は、Google Cloudパートナー企業に所属する技術者を対象としたアワードです。技術力やソリューション構築力、社内外への技術貢献、クラウド技術への情熱などを総合的に審査し、優れた技術者が選出されます。

認定プログラムの詳細：
[Google Cloud Partner Top Engineer 2026 アワードプログラムのご案内](https://cloud.google.com/blog/ja/topics/partners/2026-google-cloud-partner-top-engineer-award-program?e=48754805)
（2025年6月12日にGoogle Cloud Japanのブログで発表）

今回選出されたのは、当社デジタルプラットフォーム事業本部プロダクト＆テクノロジー部 データインテリジェンスグループ データエキスパートの古畑 和輝 です。
古畑は、Google Cloudを活用したデータ基盤構築、生成AIプロジェクト支援、AIレポート自動化など数多くの業務領域において、技術面からの支援と実装を牽引してきました。
また、社内外における技術共有・登壇・コミュニティ活動にも積極的に取り組み、Google Cloudの普及と技術者育成に貢献しています。

MBKデジタルは「データとAIの力で志をカタチに」をミッションに掲げ、Google Cloudを活用した企業のデジタル変革を支援しています。
今回の選出を励みに、技術力の強化と実践的なソリューション提供を通じ、お客様の価値創造に引き続き貢献してまいります。

【古畑 和輝 コメント】　
「2年連続で選出していただき、大変光栄です。日頃の活動を評価していただけたことを嬉しく思います。
現状に満足することなく、今後も技術力の研鑽に励み、お客様への価値提供とコミュニティへの貢献に努めてまいります」
